# An English Language Internalisation & Activation Project
> English is not a subject to master, but a craft to practise gently every day.

Key assignments

## 1. Read, not too much
 - personal essays: blog
 - scinece writing for non-experts
 - memoirs in plain prose
 - opinion columns
 - NO fully academic texts
 - highlight phrases, not words
 - focus on sentence openers, transitions

## 2. Establish a [phrase bank](/phrases.md)
> it is totally fine to be awkward
- 150-400 words
- structure 
  - opener: what I find interesting
  - soft opinion: I tend to think
  - hedging: in some context
  - causal connectors: that said
- force using 5-10 items every week in speech or writing

## 3. Daily Writing
- topics: from reading, a daily observation, minor frustration, casualy explained scientific idea
- write as if explaining to a friend at a caf√©
- write short sentences
- don't use: I think,
- try opening expressions: Recently, I have been noticing that

## 4. Weekly Rewrite
- imitate the structure from a paragraph from reading

## 5. Retrieval speech
- Summarise reading out loud
- Force yourself to use new phrases
